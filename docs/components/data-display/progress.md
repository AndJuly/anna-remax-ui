## API

| 属性    | 说明           | 类型                           | 默认值 |
| ------- | -------------- | ------------------------------ | ------ |
| percent | 百分比         | number                         | -      |
| color   | 进度条颜色     | string                         | -      |
| bgColor | 进度条底色     | string                         | -      |
| size    | 进度条大小     | `small` \| `middle` \| `large` | -      |
| shape   | 进度条形状     | `round` \| `square` \| `sharp` | -      |
| width   | 进度条线的宽度 | number \| string               | -      |
